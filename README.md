# Twilight-Forms
A Minecraft Cobblemon datapack that adds 50+ custom Twilight Forest forms and over 250+ spawns to the Twilight Forest dimension. Requires the Cobblemon and Twilight Forest mods.

Official documentation: https://docs.google.com/spreadsheets/d/1AafuQTbn8ZOYbnxcc8pOjfavEZGB1dysjJn2TxD4Bww/edit?gid=1915767084#gid=1915767084


Use in-game command   '/pokedex grant all twilight'   to view all the new Pokemon added to the Twilight Forest dimension

Discord thread post: https://discord.com/channels/934267676354834442/1221575822759170058

Modrinth page: https://modrinth.com/datapack/twilight-forest-forms-cobblemon

For suggestions, post on the discord thread here: https://discord.com/channels/934267676354834442/1195373518095921233

Link to Github repo: https://github.com/kingtrapinch/Twilight-Forms
